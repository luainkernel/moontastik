Module l4.udp

UDP Header Parsing and Packing Module This module provides utilities for parsing, packing, and manipulating UDP headers.

It includes functions for handling UDP packets, calculating lengths, and managing checksum fields.

### UDP Header Structure ``` UDP Header { spt (16): Source Port. dpt (16): Destination Port. len (16): Length (header + payload, in bytes). checksum (16): Checksum for error detection. payload (variable): Payload data. } ```

References: - RFC 768: User Datagram Protocol (UDP)

Functions

pack (self)
Packs the UDP header and payload into a binary string. Calculates the total length of the UDP packet and constructs the binary representation.

Parameters:

  • self table The UDP header object.

Returns:

    string Binary string representing the packed UDP header and payload.
parse ([off=1])
Parses a binary string into a UDP header structure. Extracts the source port, destination port, length, and checksum from the binary string.

Parameters:

  • off number Offset to start parsing from. Defaults to 1. (default 1)

Returns:

  1. table Parsed UDP header as a table.
  2. number The next offset after parsing.
