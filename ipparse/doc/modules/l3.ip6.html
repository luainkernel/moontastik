Module l3.ip6

IPv6 Header Parsing and Packing Module This module provides utilities for parsing, packing, and manipulating IPv6 headers.

It includes functions for handling IPv6 addresses, converting between binary and readable formats, and managing IPv6 network addresses.

### IPv6 Header Structure ``` IPv6 Header { version (4): IP version (always 6 for IPv6). traffic_class (8): Traffic Class. flow_label (20): Flow Label. payload_len (16): Payload Length (in bytes, excluding the header). next_header (8): Next Header (e.g., TCP, UDP, or extension headers). hop_limit (8): Hop Limit (similar to TTL in IPv4). src (128): Source IPv6 Address. dst (128): Destination IPv6 Address. payload (variable): Payload data (e.g., transport-layer data or extension headers). } ```

References: - RFC 8200: Internet Protocol, Version 6 (IPv6) Specification
