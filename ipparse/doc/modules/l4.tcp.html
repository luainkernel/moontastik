Module l4.tcp

TCP Header Parsing and Packing Module This module provides utilities for parsing, packing, and manipulating TCP headers.

It includes functions for handling TCP packets, managing flags, and calculating offsets.

### TCP Header Structure ``` TCP Header { spt (16): Source Port. dpt (16): Destination Port. seq_n (32): Sequence Number. ack_n (32): Acknowledgment Number. header_len (4): Header Length (in 32-bit words). flags (6): Flags: - 0x01: FIN (Finish). - 0x02: SYN (Synchronize). - 0x04: RST (Reset). - 0x08: PSH (Push). - 0x10: ACK (Acknowledgment). - 0x20: URG (Urgent). window (16): Window Size. checksum (16): Checksum for error detection. urg_ptr (16): Urgent Pointer. options (variable): Optional fields (if any). payload (variable): Payload data. } ```

References: - RFC 793: Transmission Control Protocol (TCP)
